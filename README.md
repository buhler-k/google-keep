# Google-Keep Clone Project 

  
  Interactive Google Keep Clone built with HTML, CSS and JavaScript with basic interactive uses including the following:

  * Creating notes (Title, Text)

    To create notes, click on the populated form, a modal form for specifically creating notes will be displayed. The form that will then be displayed allows for Title and Text creation in the modal form.

  * Editing Notes 

    In an event where you have created a note and you want to make modifications to the note, hover over the created note and click on it, the modal will enlarge and you will be able to edit either the title or text or even both. 

  * Closing Notes

    In an event you change your mind on creating the form you can simply click on the "close" button and the note details will be rendered null.

  * Archiving Notes(Title, Text)

    For created notes, those that you want to delete/archive, you can hover over the created note and click on "archive" and your note will be removed from the screen.
    
  * Sidebar(Non-Interactive)

    The sidebar is not interactive, however, you can hover over it and you will see the various possible interactive uses of the app in the case where the site is fully interactive for the users.


    ##Running the Project

    Clone the repository to your local terminal and open with live server
